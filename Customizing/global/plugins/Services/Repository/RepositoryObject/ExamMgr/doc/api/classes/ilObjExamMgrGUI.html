\ilObjExamMgrGUI

Main user interface class.

All actions of the plugin are started via HTTP requests that are handled by the performCommand method of this class. It performs access checking and then calls the desired method(s).

The user interface comprises multiple tabs: one for the client to see the current status of the plugin (scheduled runs, enrolled students, ...) and several tabs for admins to fulfill management tasks.

The ilCtrl_* annotations are copied from the example Plugin, ILIAS dev documentation
