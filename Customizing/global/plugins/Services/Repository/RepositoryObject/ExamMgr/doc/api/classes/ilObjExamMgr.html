getTicket()

getTicket() : \RTTicket

Get ticket for this exam.

Since the usage of the ticket system can be changed, it is possible that there is no ticket id for an exam; then a \ilExamMgrDummyTicket is returned. Or there is a ticket id, but the ticket system is disabled right now; then the \ilExamMgrTicket class is responsible to ignore all actions.

Returns

\RTTicket —

Either a "real" or a dummy ticket object.
